<!DOCTYPE html> — указывает браузеру, что используется стандарт HTML5.
<html lang="ru"> — тег <html> является корневым элементом страницы. Атрибут lang="ru" указывает, что язык содержимого — русский.
<head> — секция, содержащая информацию о странице (например, мета-данные и подключение стилей).
<meta charset="UTF-8"> — определяет кодировку символов как UTF-8, которая поддерживает большинство языков мира.
<meta name="viewport" content="width=device-width, initial-scale=1.0"> — помогает адаптировать страницу для мобильных устройств, делая её отзывчивой.
<title>Kusk Archive</title> — задаёт заголовок страницы, который будет отображаться на вкладке браузера.
<link rel="stylesheet" href="styles.css"> — подключает внешний CSS-файл для стилизации страницы.

<body> — содержит всё, что будет отображаться на странице.
<div class="header"> — это блок-контейнер с классом header, который используется для шапки сайта.
<h1>Kusk Archive</h1> — заголовок уровня 1 (самый крупный), используется для названия сайта.

<div class="container"> — контейнер для боковой панели и основного контента.
<div class="sidebar"> — блок с классом sidebar, представляющий боковую панель навигации.
<h2>Страницы</h2> — заголовок уровня 2 для раздела с навигацией.
<ul> — открывает список. Списки могут быть маркированными (ul) или нумерованными (ol).
<li><a href="#">Minecraft</a></li> — элемент списка (li). Внутри элемента ссылка (a), которая ведёт на другую страницу. Символ # означает, что ссылка пока никуда не ведёт.

Продолжение блока sidebar с дополнительными заголовками и списками ссылок.

<div class="content"> — блок для основного контента сайта.
<h2>Добро пожаловать в сайт Kusk Archive</h2> — заголовок уровня 2 для приветственного текста.
<p><strong>Новая версия сайта!</strong></p> — параграф (p), в котором содержится текст с выделенным жирным шрифтом (strong).
Оставшийся текст — это абзацы с описанием сайта.

CSS-файл

body — стили для всего тела страницы.
font-family: 'Arial', sans-serif; — шрифт для текста на странице.
background-color: #e0e5f0; — задаёт светло-серый цвет фона.
margin: 0; — убирает отступы вокруг страницы.
padding: 0; — убирает внутренние отступы внутри страницы.

.header — стили для шапки сайта.
background-color: #3a6ea5; — синий фон.
color: white; — белый цвет текста.
padding: 20px; — внутренний отступ (пространство между текстом и краем блока).
text-align: center; — выравнивание текста по центру.
font-size: 24px; — размер текста.

.container — это контейнер, внутри которого элементы располагаются с помощью флексбоксов.
display: flex; — делает блоки внутри контейнера горизонтальными.

.sidebar — стили для боковой панели.
background-color: #cfe2f3; — светло-синий фон.
padding: 20px; — внутренний отступ.
width: 250px; — ширина боковой панели.

.content — стили для блока основного контента.
padding: 20px; — внутренний отступ.
background-color: white; — белый фон.
flex-grow: 1; — этот элемент расширяется, занимая оставшееся пространство.

<img src="minecraft-icon.png" alt="Minecraft" class="icon"> — вставляет изображение с помощью тега <img>.
src="minecraft-icon.png" — путь к картинке (может быть как локальный файл, так и URL).
alt="Minecraft" — альтернативный текст, который отображается, если картинка не загрузится.
class="icon" — CSS-класс, который мы используем для стилизации и настройки размеров изображения.

width: 20px; и height: 20px; — задаёт размеры изображения (ты можешь изменить размеры по своему усмотрению).
vertical-align: middle; — это выравнивает изображение по вертикали относительно текста, чтобы они располагались на одной линии.
margin-right: 10px; — добавляет отступ справа, чтобы текст не прилипал к изображению.

list-style-type: none; — отключает стандартные маркеры списка (точки, кружки и т.д.).
padding-left: 0; — убирает отступы перед элементами списка. Это нужно, чтобы список был ровно выровнен по краю боковой панели.
